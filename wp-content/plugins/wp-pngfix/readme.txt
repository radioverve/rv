Plugin that fixes the png image transparency issue on IE.

== Description ==

This plugin uses CSS technique to fix the png image transparency issue and jQuery to apply this script to all of png images shown in the page.
This doesn't work for positioned and tiled png background images. Maybe in next versions.

== Installation ==

1. Upload 'wp-pngfix' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Thats all. Go to your main page and check if there's any errors.

Note : Your theme MUST have included `<?php wp_head()?>` in its header file.
