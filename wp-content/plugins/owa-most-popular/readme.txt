== Description ==
Adds a sidebar widget to display the most popular posts. The stats are generated on the data provided by the OWA plugin.  There are three algorithms which you can choose:
* Popular : the most page visits for a given date range
* Weighted : the most page visits divided by the age of a post
* Live : shows the latest page views
